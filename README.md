# SDTS 2026 Teaching Achievement Showcase

A teaching achievement exhibition platform built with Vue 3 + Vite + Element Plus.

## Tech Stack

- **Framework:** Vue 3 (Composition API)
- **Build Tool:** Vite 5
- **UI Library:** Element Plus
- **Icons:** @element-plus/icons-vue
- **Router:** Vue Router 4
- **Preprocessor:** Sass

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```
